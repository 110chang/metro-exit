About

Dokokara.infoはどこから（東京メトロの駅に）入るのが近いかわかるアプリです。東京メトロAPIの地物情報から出入口の情報を取得します。

入力した住所、もしくはデバイスの位置から一定の範囲にある東京メトロの出入口を検索できます。

本アプリは以下の外部のAPIを利用します。

  • 東京メトロAPI
  • Google Maps API
  • Google Directions API
  • Google Geocoding API

本アプリは各ブラウザの最新版で動作を確認しております。

※Android標準ブラウザはAndroid 4.1にて動作を確認しております。

Terms of use

東京メトロAPIの地物情報に登録されている以外の出入口は検索できませんのでご了承ください。

ルート表示は住所、あるいは緯度・経度から検索した目安です。実際の建造物の出入口が近いとは限りませんので、現地の状況をよく確認の上ご利用ください。

本アプリを利用したことによる、いかなる損害も負いかねます。
